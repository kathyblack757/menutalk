# 菜根谭 MenuTalk — 设计规范

## 视觉风格

### 设计理念
简洁、直观、轻量 —— 用户在小餐馆面对陌生菜单时已经很焦虑了，界面需要提供安全感。

### 色彩系统

| 角色 | 色值 | 用途 |
|------|------|------|
| 主色（金） | `#C9A96E` | 按钮、图标、强调元素 |
| 深棕 | `#8B6F47` | 标题文字、重要信息 |
| 浅棕 | `#D4C4A8` | 卡片背景、次要区域 |
| 米白 | `#FBF7F0` | 页面背景 |
| 深灰 | `#3D3D3D` | 正文文字 |
| 浅灰 | `#9CA3AF` | 辅助文字、提示 |
| 红色 | `#E74C3C` | 过敏原警告 |
| 绿色 | `#27AE60` | 购物车、确认操作 |

### 字体

- 中文：系统默认字体（-apple-system, PingFang SC, Microsoft YaHei）
- 英文/数字：Inter
- 字号层级：标题 20px / 正文 15px / 辅助 12px

### 圆角与阴影

- 卡片圆角：16px
- 按钮圆角：12px
- 卡片阴影：`0 2px 12px rgba(0,0,0,0.08)`
- 底部导航：顶部细线 + 轻微阴影

## 页面布局

### 1. 拍摄Tab（默认页）
- 中央大按钮：圆形相机图标
- 底部提示文字："拍摄菜单，即刻翻译"
- 支持从相册选取

### 2. 卡片浏览页
- 全屏高透底图（用户拍摄照片，opacity: 0.15）
- 菜品卡片居中，宽度约85vw
- 顶部左侧：返回按钮
- 顶部中央：进度 "3/15"
- 卡片下方：操作提示 "← 想要" "跳过 →"

### 3. 购物车
- 右下角浮动按钮，显示数量角标
- 点击弹出抽屉/半屏面板
- 菜品列表 + 单价
- 底部：生成点单文案按钮
- 总价显示 + 汇率换算结果

### 4. 点单文案页
- 当地语言文字 + 中文翻译对照
- 大号朗读按钮（🔊图标）
- 总价信息卡片

### 5. 历史Tab
- 时间线列表布局
- 每条记录：缩略图 + 日期 + 菜品数量
- 点击进入详情

### 6. 个人中心Tab
- 设置表单：语言选择、货币选择
- 饮食禁忌多选：花生、牛奶、鸡蛋、海鲜、大豆、小麦、坚果
- 底部：关于信息

## 交互动画

- 卡片滑动：跟随手指平移 + 旋转（±15°）
- 左滑飞出：卡片飞向左上角，缩小消失
- 右滑飞出：卡片飞向右上角，缩小消失
- 购物车弹窗：从底部滑入
- Tab切换：无过渡（瞬时切换）
